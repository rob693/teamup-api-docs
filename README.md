This repository contains a fully structured, developer-friendly, AI-optimised documentation set for the **TeamUp API**.

The purpose is to provide a reliable, search-friendly API dataset for:

- Internal systems development  
- n8n workflow automation

- API research and analysis

## üõ† How This Documentation Was Generated

The pipeline that produced this repo:

1. Raw TeamUp HTML docs were supplied from a full scrape.  
2. HTML was cleaned into plaintext.  
3. A Python extractor script applied heuristic extraction (Mode B) to identify:
   - HTTP method  
   - URL  
   - JSON request/response blocks  
   - Error codes  
   - Summary lines  
4. Endpoints were categorised using a broad normalisation system (e.g. all discount code endpoints ‚Üí `discount-codes` folder).  
5. Clean Markdown files were generated for each endpoint.  
6. A structured folder hierarchy + index.json was created.

This process ensures both **accuracy** and **LLM readability**.
